## 前言

本项目初衷为解决SCAU收集试卷难，零散，期末才能拿到试卷等等问题，对于像大卷特卷的选手来说十分这是问题困扰他们很久。对于一些突击的同学而言，找资料难，不全面也是一大难题。

前辈们其实也有做过类似的尝试，但是他们的想法局限于自建服务器和存储数据，反而忽视了公共资源的使用。

加之还有文件审核的问题，故特意放到 Github 之上能够提高发布的门槛，减少审核上的问题。

后续也会继续利用Github Page建立索引库，降低下载资料的门槛。

无论如何，希望看到这里的你能成为Contributor，为现人及后人出一份力

## 规范

每个目录需要有README.md，在里面填写更新内容，并且在Contributor栏署名，邮箱，首次贡献时间，留下你的一笔。

并且在更新栏填写新的内容

### 文件夹命名

如无特殊情况（特殊情况包括含领导人名字，生僻字）请使用**中文全称**来命名。如下为案例。 

- $\cancel{高数} \Rightarrow 高等数学$ 
- $\cancel{国安} \Rightarrow 国家安全教育$ (其实没有这门课)

 

如果需要英文标注，多个单词之间请使用 - 进行连接，单词首字母需要大写(介词除外)

如：

- Introduction-to-Probability
- Linear-Algebra

### 文件命名

在专门的课程下建立“**Exam**”文件夹，上传原试卷和试卷压缩包**课程名称试卷.zip**

每次push新的试卷提pr请帮忙压缩一份所有文件的压缩包十分感谢！
